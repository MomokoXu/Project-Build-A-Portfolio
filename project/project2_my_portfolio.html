Book For You Recommender System

BYOU, Book For You book recommender system, is a web application which takes the user's id as input and output recommended books for the user.This book recommender system consists of both front-end and back-end implementations. The core part for the recommender are four recommendation algorithms. Three of them are implemented by Mahout, that is item-based recommendation, user-based recommendation, and single value decomposition (SVD) recommendation. The rest one is imple- mented from scratch, which is a contend-based algorithm using TF-IDF.

Emate: Friend Finder

Emate is an application that enables users to find people who share the most similar thoughts with him/her in real time. Users will be able to enter one or more phrases as he/she likes into the application, and then get feedback from the application which will look into the historical data in the near past hours, score the relevancy and geological closeness of every single tweet, and output to the user with users who have the highest similarities and are located nearest to the user. Then the user would be provided the corresponding username and the actual contents of the tweets, so that he/she can choose to establish connection with the ones as wanted.

Movie Trailers Web App

This mini project aims to provide server-side code to store a list of author's favorite movies with box art imagery and a movie trailer URL so that front-end web page (provided by Udacity Full Stack Nano Degree program) uses this data for visitors to review the movies and trailers.
